📊 Análisis de Ventas con Python y Power BI
🧠 Descripción

Este proyecto consiste en un análisis de datos de ventas utilizando Python para la limpieza y Power BI para la visualización.

El objetivo fue identificar patrones, tendencias y oportunidades de mejora en el negocio a partir de los datos.

🛠️ Herramientas utilizadas
Python (pandas)
Excel
Power BI

Principales insights
La categoría Furniture lidera en ventas
Algunos productos concentran gran parte de los ingresos
Las ventas están distribuidas entre varias ciudades
Se observan variaciones en el comportamiento de ventas en el tiempo

Conclusión
El análisis permitió identificar patrones clave en el comportamiento de ventas, lo que puede apoyar la toma de decisiones estratégicas en el negocio.
